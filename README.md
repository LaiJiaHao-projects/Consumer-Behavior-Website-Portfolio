# Senior Healthcare Insights Portfolio

An interactive, responsive Single Page Application (SPA) showcasing a comprehensive consumer behavior analysis of Malaysia’s aging demographic and strategic marketing recommendations for healthcare providers.

## 🚀 Overview
This repository contains the front-end source code for the **Senior Healthcare Insights Portfolio** developed as a capstone project for *MKT2054 Consumer Behaviour*. The application presents demographic projections, health profile metrics, theoretical marketing case studies, and strategic recommendations for engaging senior healthcare consumers in Malaysia.

## 🛠️ Key Features
- **Dynamic Tab Navigation:** Seamless switching between market overviews, segment analyses, case studies, recommendations, and full citations.
- **Interactive Data Visualization:** Custom JavaScript-driven timeline dynamic bar charts tracking age structure shifts from 1970 to 2040.
- **Interactive Accordions:** Collapsible UI elements for seamless reading of case studies and references.
- **Modern Responsive Design:** Built using **Tailwind CSS** and styled with **Google Fonts** (*DM Sans* & *Playfair Display*) and **Lucide Icons**.

## 💻 Tech Stack
- **HTML5** & **Vanilla JavaScript (ES6+)**
- **Tailwind CSS v3.4** (Utility-first styling)
- **Lucide Icons** (UI Icons)
